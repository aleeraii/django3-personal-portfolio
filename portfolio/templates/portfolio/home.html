COVID-19 Analyzer

A console application that represents the world data of cases related to COVID-19. The data taken is from official COVID-19 Api. As the data is updated on the api daily so the results shown are till date. The user can view the statistic of either a single country or compare results of two countries.

Subnet Calculator

A Subnet Calculator that simple takes in the Ip Address and gives out the results related to that Ip Address such as it's subnet mask, number of hosts, binary form of subnet mask etc.

Threaded Port Scanner

A port scanner that uses the threads to scan the available free or used ports of the given website. It shows a graph of how much time it takes to scan all the given ports using different number of ports.
